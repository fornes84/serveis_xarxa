Text de la pregunta

Emparella les següents definicions

El servidor que proporciona respostes sobre les zones que té configurades és

El servidor que reenvia les consultes a altres servidors és

El servidor que realitza diverses consultes fins a obtenir una resposta completa és

El servidor DNS que conté els fitxers de zona (la base de dades original) és

El servidor DNS que rep la base de dades de zona d’un altre servidor és

Retroacció

La resposta és correcta.

La resposta correcta és: El servidor que proporciona respostes sobre les zones que té configurades és → Authoritative, El servidor que reenvia les consultes a altres servidors és → Forward, El servidor que realitza diverses consultes fins a obtenir una resposta completa és → Recursive, El servidor DNS que conté els fitxers de zona (la base de dades original) és → Primary, El servidor DNS que rep la base de dades de zona d’un altre servidor és → Secondary.

Pregunta 12

Parcialment correcte
Puntuació 0,60 sobre 1,00
Marca la pregunta

Text de la pregunta

Emparella els següents camps del registre SOA amb la seva funció:

Temps que els servidors secundaris deixen passar abans de consultar el servidor primari si ha passat algun canvi a la zona.

El temps que els solucionadors han de guardar a la memòria cau una resposta de domini o host inexistent.

Si passat aquest temps, un servidor secundari no aconsegueix realitzar la transferència de zona, ha de deixar de considerar-la vàlida.

Temps que un servidor secundari deixa passar abans de tornar a intentar una transferència de zona.

Utilitzat pels servidors secundaris per a detectar els canvis a la base de dades

Retroacció

La resposta és parcialment correcta.

Heu seleccionat correctament 3.
La resposta correcta és: Temps que els servidors secundaris deixen passar abans de consultar el servidor primari si ha passat algun canvi a la zona. → time-to-refresh, El temps que els solucionadors han de guardar a la memòria cau una resposta de domini o host inexistent. → nxdomain-ttl, Si passat aquest temps, un servidor secundari no aconsegueix realitzar la transferència de zona, ha de deixar de considerar-la vàlida. → time-to-expire, Temps que un servidor secundari deixa passar abans de tornar a intentar una transferència de zona. → time-to-retry, Utilitzat pels servidors secundaris per a detectar els canvis a la base de dades → serial-number.
